This site uses Hugo + Hextra theme

Features: 

- Multilingual
- Full control in organising content
  - Ability to write next and previous sections, to signpost relevant sections.
- Glossary: to have inline definitions like this one about {{< term "fediverse" >}} here. See:  https://imfing.github.io/hextra/glossary/
- Blog-like template: 
- Reusing content
- Uses Tailwind, like bonfire.
- can be deployed automatically using github actions: 

> [!note]
> You can read the full list of features in the official Hexa documentation site: https://imfing.github.io/hextra/docs/ and this release note: https://imfing.github.io/hextra/blog/v0.12/
